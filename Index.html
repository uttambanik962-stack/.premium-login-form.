DESCRIPTION -------------- This is a modern, creative, and lightweight login form built with pure HTML, CSS, and Vanilla JavaScript. The unique feature of this form is the "Light Beam" or "Flashlight" effect that activates when the user toggles password visibility. It also includes a smooth transition between Light and Dark modes. 2. KEY FEATURES --------------- * Unique Magic Light Beam Effect on password field. * Smooth Light/Dark Mode toggle. * Minimalist and Premium UI design. * Built-in Form Validation (Email & Password). * Fully Responsive (Works on Mobile, Tablet, and Desktop). * No External Libraries/Frameworks (No jQuery or Bootstrap needed). * Clean and well-commented code. 3. WHAT IT DOES (WORKING FEATURES) ---------------------------------- * Password Visibility: Clicking the "Eye" icon reveals the password and triggers the flashlight effect. * Dark Mode: Toggling the eye icon switches the entire page to a dark theme. * Validation: If fields are empty during submission, the input borders turn red automatically. * Input Feedback: Red error borders disappear instantly when the user starts typing. * Customization: Easy to change colors using CSS variables (:root). 4. WHAT IT DOES NOT DO (LIMITATIONS) ------------------------------------ * Backend Integration: This is a Front-end template only. It does not include PHP/Node.js or Database (MySQL) integration for actual user login. * Email Format Validation: Currently only checks if the field is empty. It doesn't check for "@" or "." symbols (can be added easily). * Password Strength Checker: It doesn't analyze if the password is "weak" or "strong". 5.
